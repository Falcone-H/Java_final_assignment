# Java final assignment
The java final assignment

## 前端
1、登录页(`login_page.class`)
* 输入账号
* 输入密码
* 登录按钮、注册按钮

2、主页面(`main_page.class`)
* 用户名
* 性别
* 编号
* 简介
* 头像
* 显示所有用户的信息

## 后端
1、数据库配置(`druid_util.class`)
* 提供连接接口
* 提供关闭接口

2、数据库操作(`Executor.class`)
* 密码验证
* 信息查询
* 头像上传

3、用户类(`User.class`)
* 用户名
* 账号（唯一识别）
* 密码
* 头像
* 性别
* 编号（主键）
