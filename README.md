# Java final assignment
The java final assignment

## 前端
1、登录页(`login_page.java`)
* 输入账号
* 输入密码
* 登录按钮、注册按钮

2、主页面(`main_page.java`)
* 用户名
* 性别
* 编号
* 简介（未实现）
* 头像（未实现）
* 显示所有用户的信息

3、注册页(`register_page.java`)

* 用户名
* 账号
* 密码
* 性别

## 后端
1、数据库配置(`druid_util.java`)
* 提供连接接口
* 提供关闭接口

2、数据库操作(`Executor.java`)
* 密码验证
* 信息查询
* 头像上传

3、用户类(`User.java`)
* 用户名
* 账号（唯一识别）
* 密码
* 头像
* 性别
* 编号（主键）
